  1. גודל התוכן הקובע את גודל הבסיס של האלמנט. המאפיינים width ו height ב CSS מאפשרים לציין את גודל התוכן לאלמנט בצורה מפורשת (גם אם המספר שבחרתם לא מתאים לגודל האמיתי של התוכן).
  2. גודל הריפוד: ריפוד הוא עטיפה סביב לאלמנט בתוכה לא ייכתב תוכן אך בעלת אותו צבע הרקע כמו האלמנט. מטרת הריפוד לאפשר ריפוד פנימי בתוך אלמנט או בין מיכל לאלמנטים הפנימיים שלו. המאפיין padding וכן המאפיינים padding-top, padding-bottom, padding-left, padding-right קובעים את גודל הריפוד של אלמנט.
  1. גודל הגבול: גבול הוא שכבה נוספת המצוירת מסביב לריפוד ויכולה לקבל צבע נפרד. הגבול משמש להפרדה בין האלמנט לסביבה החיצונית. המאפיין border וכן המאפיינים border-top, border-bottom, border-left, border-right, border-color, border-style קובעים את מאפייני הגבול של האלמנט.
  2. ודל השוליים: שוליים הם המרחק בין שני אלמנטים. המאפיין margin וכן המאפיינים margin-left, margin-top, margin-bottom ו margin-right קובעים את גודל השוליים סביב אלמנט
